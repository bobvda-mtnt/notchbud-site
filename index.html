Works with any AI

Bring your own AI.

Claude, ChatGPT, a cron job: anything that can write a text file can power NotchBud. Copy this block into your scheduled task, and you're set.

Copy 
=== DELIVER TO NOTCHBUD ===

Write one markdown file per day into the NotchBud folder.

1. File name: YYYY-MM-DD.md, first line: # Your Title
2. Sections: ### 🔴 Urgent, ### 📅 Today's Schedule, …
   each becomes a collapsible card
3. One item per "- " bullet, each becomes swipeable
4. Tasks: ### ✅ Action Items with "- [ ] task" lines
   → they import into the todo list
5. Read feedback.json first: never repeat "cleared" items,
   surface "prio" items on top, plan around "reminders"
6. Evening runs: APPEND a ### 🌙 section, never rewrite

=== END NOTCHBUD ===

FAQ

Questions.

Do I need a Mac with a notch?

NotchBud is designed for notched MacBooks (2021 and later). A menu-bar mode for external displays and older Macs is on the roadmap.

Which AI does it work with?

Any of them. NotchBud reads plain markdown from a folder: Claude scheduled tasks, ChatGPT, n8n, a cron script, or you typing by hand. If it can write a file, it can brief you.

Where does my data live?

In a folder on your own Mac, as markdown and JSON you can open with any editor. A local folder is the default and the safest. NotchBud has no servers, no analytics, no account.

Will it fill up my disk?

No. NotchBud keeps 30 days of briefings and automatically expires stale flags. A full year of daily use is a few hundred kilobytes.

What happens after the trial?

NotchBud asks for a license. Your briefings and todos stay untouched in your folder either way. It is your data.

Is there an iPhone app?

In development: a companion app with home-screen widgets that syncs your todos through your own iCloud. It will be free for license holders when it ships.
